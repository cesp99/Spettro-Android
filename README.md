## Spettro

Android client for Spettro (by Eyed). Connects to a Spettro backend/service to exchange events, send commands, and chat.

### Features
- Real-time event streaming and status monitoring
- Chat interface with command sheet
- Persistent connection preferences
- Compose UI with Material 3

### Tech Stack
- Kotlin + Jetpack Compose
- AndroidX (Navigation, Lifecycle, ViewModel)
- Gradle (KTS) with version catalog

### Build & Run
```bash
./gradlew assembleDebug
```
Install the resulting APK on an Android device/emulator (API 24+ recommended).

### Package
`de.aploi.spettrobyeyed`
